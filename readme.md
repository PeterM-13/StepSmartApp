# 🧑‍🦯 Step Smart App

React Native app for IMechE project.

App that connects to smart walking stick product, to control the stick and display metrics collected by the stick.

## 📂 Dependencies
- npm
- expo
- react-native
- native-stack
- react-native-screens react-native-safe-area-context

## ▶️ Deployment
Run: npm start
